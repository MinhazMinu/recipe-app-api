###### Recipe Api design

-   Features
    -   Create
    -   List
    -   view details
    -   Update
    -   Delete

*   Endpoints
    -   /recipes/
        -   GET - List all recipes
        -   POST - Create a new recipe
    -   /recipe/<recipe_id>
        -   GET - view details of a recipe
        -   PUT/PATCH - Update a recipe
        -   DELETE - Delete a recipe
